# Application Process

```
  +-------------------+
  |   User provides   |
  |      a URL        |
  +---------+---------+
            |
            v
   +---------------------+
   | 1. Fetch the URL    |
   | 2. Save data        |
   | 3. Find next link?  |
   +---------------------+
            |
      +-----+-----+
      |           |
     Yes         No
      |           |
      v           v
  +-------------------+    
  | Fetch next URL   |    
  +-------------------+    
      |           |
      +-----<-----+
            |
            v
   +-----------------------------------------------+
   | Is fetching ok and should we launch summarization? |
   +-----------------------------------------------+
            |
            v
   +-----------------------------+
   | Summarize all pages         |
   |   (in reverse order)        |
   +-----------------------------+
            |
            v
   +-------------------+
   |  Return result    |
   +-------------------+
```
